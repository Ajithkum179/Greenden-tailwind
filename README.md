🌿 Greenden - Plant Shop Website

Greenden is a simple and responsive plant shop website built using HTML and CSS. This project showcases a modern UI design for an online plant store where users can explore plants, view product sections, and contact the store.

🚀 Features

Responsive website layout

Clean and modern UI design

Home page with featured plants

Product section displaying different plants

About section

Contact section

Simple navigation bar

Footer with social links

🛠️ Technologies Used

HTML5

CSS3

Flexbox

Google Fonts
